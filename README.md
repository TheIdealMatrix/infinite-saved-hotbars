# Infinite Saved Hotbars

A Minecraft mod that extends the Creative Saved Hotbars infinitely.

## Description

Ever wished for unlimited Creative Saved Hotbars space? This mod makes it a reality! The Infinite Saved Hotbars mod allows you to store an unlimited number of items in your Creative Saved Hotbars.

## Features

* Endless storage for your favorite items
* Easy item management:
	+ Add an item by clicking inside a slot while holding it.
	+ Override an item by clicking inside a slot while holding down **Ctrl**.
	+ Empty a slot by clicking on it while holding **Ctrl**.

## Installation

1. Download the mod from [Modrinth](https://modrinth.com/mod/infinite-saved-hotbars#download).
2. Place the mod file in your `.minecraft/mods` directory.

## Data Storage

* Your infinite hotbar data is saved in `.minecraft/hotbar_extended.nbt`
